
# Two Factor Project:

Two Factor project é um projeto idealizado por mim, onde ele tem o papel de me desafiar fazer um login e cadastro que ultilizem uma criptografia própria. Além de fazer um sistema de autenticação de dois fatores.

 


## Funcionalidades

- Temas dark e light
- Login
- Cadastro
- Sistema de criptografia próprio
- Sistema de esqueceu a senha
